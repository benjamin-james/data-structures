
* project 2

To test, run:
=go test ./...=

To build =benJames=, run
=go build=

To install, type
=go install=

To run, (after it is installed), type
=benJames input=
to read from the file =input= and produce the files 
=input.uni=, =input.bi=, =input.cp=, and =input.time=.

By default, =benJames= reads from stdin.
